Weather Database

The MWA Weather Scraper is an automated system I built to collect daily weather data from 200+ publicly available sources across Queensland and select locations in New South Wales, developed during my internship at MWA Environmental. Using Python, MySQL, and Apache, it scrapes weather info from various stations, processes it, and stores it in a local database hosted on a private server. I designed this tool to eliminate manual data collection, providing efficient access to real-time weather insights.. It updates itself daily, keeping things fresh without piling up historical records.

Relevant Skills

Python MySQL Apache Batch Script Project Management

Outcomes

The Weather Scraper now reliably collects and stores daily weather data from hunderds or reports, delivering a valuable resource for MWA engineers. The system runs autonomously, updating the database each day with minimal oversight. Error logs confirm its stability, with few issues post-setup. By automating data collection, the project saves time and ensures engineers have immediate access to current weather insights via the Apache server.
