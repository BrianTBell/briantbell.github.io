Weather Database

During my internship at MWA Environmental, I created a tool that automates weather data collection, enabling engineers to monitor conditions across 200+ stations with ease. This project enhanced operational efficiency by eliminating manual processes, offering valuable insights for environmental analysis. It ensures timely updates, supporting better decision-making in critical applications.

Project Overview

The MWA Weather Scraper is an automated system I built to collect daily weather data from publicly available sources across Queensland and select locations in New South Wales, developed during my internship at MWA Environmental. Using Python, MySQL, and Apache, it scrapes weather info from various stations, processes it, and stores it in a local database hosted on a private server. I designed this tool to eliminate manual data collection, providing efficient access to real-time weather insights.. It updates itself daily, keeping things fresh without piling up historical records.

My Role

I was the sole developer on this project, managing it from start to finish. I wrote Python scripts using libraries like BeautifulSoup and Requests to scrape weather data from public websites, set up and managed a MySQL database to store the data efficiently, configured an Apache server to make the database accessible over the local network, and implemented scheduled tasks via Windows Task Manager to run the scraper daily for seamless updates.

Technical Skills

  • Web Scraping: Leveraging Python to parse and extract data from dynamic web pages.
  • Database Management: Designed and maintained a MySQL database for structured data storage and retrieval.
  • Server Administration: Configured and managed an Apache server to serve the database locally.
  • Automation: Utilized Windows Task Manager to automate daily scraping tasks, enhancing efficiency.
  • Problem-Solving: Tackled challenges such as website structure changes and data consistency through robust coding solutions.

Challenges & Solutions

1. Website Changes

Challenge:The scraper relied on certain sites' HTML structure, which occasionally changes, breaking the parsing logic.

Solution:I implemented error handling and logging to detect issues quickly, allowing for rapid adjustments to the scraping scripts.

2. Data Consistency

Challenge:Weather data formats varied across weather stations, complicating storage and analysis.

Solution:I developed parsing functions to standardize data (e.g., converting dates and handling missing values) before database insertion.

3. Scalability Issues

Challenge:Scraping hundreds of weather reports in a single script caused performance bottlenecks and occasional crashes.

Solution:I split the workload into multiple Python scripts (e.g., full_scrape1 to full_scrape6), each handling a subset of stations, and scheduled them separately via Task Manager

4. Complex Installation

Challenge:Setting up Python, MySQL, Apache, and Task Manager was non-trivial for new users.

Solution:I provided detailed installation instructions in the README, covering dependencies and configuration steps to ease deployment.

Outcomes & Results

The Weather Scraper now reliably collects and stores daily weather data from hunderds or reports, delivering a valuable resource for MWA engineers. The system runs autonomously, updating the database each day with minimal oversight. Error logs confirm its stability, with few issues post-setup. By automating data collection, the project saves time and ensures engineers have immediate access to current weather insights via the Apache server.

The broader outcome of this work lies in its demonstration of adaptable, scalable strategies for thermal system optimization, applicable across industries facing energy and performance challenges. This project honed my ability to integrate simulation tools, analyze complex datasets, and address practical engineering constraints, equipping me with a versatile skill set for tackling thermal management issues. These capabilities position me to drive innovation in Tampa’s heat transfer industries, such as HVAC and energy systems, through advanced thermal solutions.
